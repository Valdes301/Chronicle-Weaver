# **App Name**: Chronicle Weaver

## Core Features:

- Campaign Initialization: Initialize a new D&D 5e campaign by providing a name, setting, and brief description to the LLM tool.
- Session Management: Create, manage, and archive individual game sessions with notes on key events, NPCs, and decisions.
- Session Import: Import pre-existing sessions to continue the campaign.
- AI Session Generation: Generate new D&D sessions based on the campaign history, previous session details, and specified parameters using the LLM tool.
- Campaign Timeline: Visualize campaign progress with a timeline of completed sessions and key events.
- NPC/Location Database: Maintain a database of NPCs and important locations encountered during the campaign.
- Adventure Log: Maintain an adventure log that allows players to summarize each session after it's completed. 

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke mystery and magic.
- Background color: Dark grey (#303030), nearly black, to provide contrast with the brighter primary.
- Accent color: Yellow-orange (#FFAB40), an analogous color to the primary on the 'left' of the color wheel; vivid enough to stand out and create contrast.
- Body font: 'Literata', serif, with a literary feel.
- Headline font: 'Belleza', sans-serif; to be paired with Literata for a touch of visual tension, suiting an app about fantasy content. Note: currently only Google Fonts are supported.
- Use fantasy-themed icons (swords, maps, scrolls) for navigation and key actions.
- A card-based layout for sessions and campaign details for easy scanning and interaction.
- Subtle animations for transitions between sessions and when generating new content.
